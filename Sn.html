Group elementsÂ¶

The group elements of are of type SnElement, which have the same methods as the Permutation class. Group elements are listed in a specific reverse insertion sort order that fits the structure of cosets and hence is well adapted to Clausen-type fast Fourier transforms on . The following shows how to extract the iâth group element and how to get the index of a particular group element.

>>> pi=G[17]
>>> print(pi)
[ 3 4 2 1 ]
>>> print(G.index(pi))
17

Conjugacy classesÂ¶

The conjugacy classes of are in bijection with the integer partitions of . Snob2 has a separate class SnCClass to represent conjugacy classes. SnCClass objects can be constructed either from the group object or directly from an integer partition.

>>> G=Sn(5)
>>> mu=IntegerPartition([3,2])
>>> cc=G.cclass(mu)
>>> print(cc)
SnCClass[3,2]

>>> cc=SnCClass(mu)
>>> print(cc)
SnCClass[3,2]

An even shorter form is the following.

>>> cc=SnCClass([3,2])
>>> print(cc)
SnCClass[3,2]

The conjugacy classes are ordered according to majorization order of their integer partitions. The Sn.index method returns the index of a given conjugacy class.

>>> G.index(cc)
2

The characters of are also indexed by the integer partitions of and can be accessed through the character method of Sn.

>>> chi=G.character([3,2])

Irreducible representationsÂ¶

The irreducible representations (irreps) of are captured by SnIrrep objects. For a given integer partition of n, the corresponding irrep can be constructed from the group object or directly from the integer partition.

>>> rho=G.irrep([3,1])
>>> print(rho)
SnIrrep([3,1])

>>> rho=SnIrrep([3,1])
>>> print(rho)
SnIrrep([3,1])

The dimension of the irrep is accessible through the get_dim() method.

>>> print(rho.get_dim())
3

All irreps in Snob2 are expressed in Youngâs orthogonal representation. The representation matrices are easy to access

>>> pi=SnElement([3,2,1,4])
>>> print(rho[pi])
[ 1 0 0 ]
[ -0 -0.5 -0.866025 ]
[ -0 -0.866025 0.5 ]

or just

>>> print(rho[3,2,1,4])
[ 1 0 0 ]
[ -0 -0.5 -0.866025 ]
[ -0 -0.866025 0.5 ]

Sn typesÂ¶

The type of a representation is an associative list of integer partitions and associated multiplicities describing what irreps a particular representation is composed of. The following shows how to set up an SnType object.

>>> tau=SnType(IntegerPartition([4,1]),2)
>>> tau.set(IntegerPartition([3,2]),1)
>>> tau.set(IntegerPartition([3,1,1]),1)
>>> print(tau)
([4,1]:2,[3,2]:1,[3,1,1]:1)
